# RK Graphics & Ads Agency

Simple portfolio website for RK Graphics & Ads Agency. This repository contains a minimal static site (index.html) showcasing services, sample projects, and contact links.

## GitHub Pages

The site is ready to be published with GitHub Pages. To publish:
1. Go to the repository Settings → Pages.
2. Under "Source", select the `main` branch and the root `/` folder.
3. Save — the site will be available at: `https://rkgraphicsandads.github.io/rksportfolio/` (may take a minute to publish).
